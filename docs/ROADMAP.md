# Humidor HQ

## Vision

Humidor HQ is a personal cigar collection management system designed to track inventory, purchases, aging, smoking history, and collection analytics.

Unlike commercial cigar apps, this application focuses on providing detailed financial and inventory insights while remaining simple to use.

---

## Version 1.0 Goals

### Dashboard
- Collection Summary
- Total Cigars
- Total MSRP
- Total Actual Cost
- Total Savings
- Number of Humidors

### Inventory
- Add/Edit Cigars
- Search
- Photos
- Quantity Tracking

### Purchases
- Singles
- 5 Packs
- Boxes
- Vendor
- Purchase Date
- MSRP
- Actual Cost

### Humidors
- Multiple Humidors
- Move Cigars Between Humidors
- Humidor History

### Smoking Journal
- Date Smoked
- Rating
- Notes
- Pairing

### Reports
- Collection Value
- Spending History
- Savings
- Inventory by Humidor
